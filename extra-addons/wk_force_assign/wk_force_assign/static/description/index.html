Stock Force Assign

The module adds the feature to validate partial deliveries in Odoo. The user can force a full or partial delivery validation if the product stock is insufficient.

Why Do You Need Force Availability Options In Transfers?

Force availability allows you to validate the delivery even if the product quantity is less than the required for the order.

You can now force full or partial availability as per need. In case of adding partial availability you can also choose to create a back order.

  • Make it efficient to process delivery in Odoo.
  • It makes it easier for the user to validate the delivery of an order.
  • Curtails the number of steps for the user making delivery validation faster in Odoo.

Detailed Features List

  • The module adds the functionality of ‘force availability’ and ‘partal force assign’ while validating delivery.
  • The Odoo user can choose to validate the delivery even when the products are not in stock with the ‘force availability’ button.
  • It also allows the user to validate a partial delivery for an order using ‘partial force assign’.
  • For partial validation the user can either create a backorder for the remaining product(s) quantities or cancel them.

Force Apply the product availability in Odoo to validate the delivery of an order.

You can also force validate partial delivery to either create backorder or cancel the remaining quantities in an order.
